make sure you have python 3.6+ installed on your machine. 
upgrade pip and venv if you wish to run in a virtual environment 

run the command below to install packages needed for this project: 
pip install -r requirment.txt

After installation is complete run the following command to start the web application on local host. 
python main.py 

navigate to 127.0.0.1:5000 on your browser to access the website.
